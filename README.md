# Software Agent Architect

An interactive dashboard visualizing the Context Engineering architecture for a Software Agent, featuring architectural layers, data flows, and a live AI simulation using the Gemini API.

## Features

- **System Context View**: Visualizes the external environment and agent interactions.
- **Layer Architecture View**: Interactive exploration of the 7-layer agent architecture.
- **Data Flow View**: Animated data flow from client request to code execution.
- **Operations Dashboard**: Real-time metrics for system health and performance.
- **Simulation**: Live chat interface with the Software Agent powered by Gemini 2.5 Flash.

## Setup

1. Clone the repository.
2. Install dependencies: `npm install`
3. Create a `.env` file and add your `API_KEY`.
4. Run the development server: `npm run dev`

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Recharts
- Google GenAI SDK
